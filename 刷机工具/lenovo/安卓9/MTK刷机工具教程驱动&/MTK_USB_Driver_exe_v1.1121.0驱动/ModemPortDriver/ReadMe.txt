ReadMe.txt

	This application supports to install a modem driver, run modeminstaller.exe to complete the installation. The modem driver including a com port driver, a co installer file and an INF file. 

[Support OS]
WinXP , Win2000 , Vista 32, Vista 64, Win7 32 , Win7 64.

[Support Device ID]
USB\VID_0E8d&PID_00A1&MI_00
USB\VID_0E8d&PID_00A2&MI_00

[Note]
1. You must make sure that device is one of the support device ID¡£
2. If you manually install a driver in before, uninstall the driver ,plug out and plug in device to make the newest driver to be installed after running modeminstaller.exe.
