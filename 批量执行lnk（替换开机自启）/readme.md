说明：
将需要批量启动的快捷方式（*.lnk）放在脚本同级文件夹下，运行脚本即可
自启方案：
将脚本放在对应目录下
win7：C:\Users\Administrator\AppData\Roaming\Microsoft\Windows\Start Menu\Programs\Startup
win10：C:\ProgramData\Microsoft\Windows\Start Menu\Programs\StartUp
启动时会执行该脚本
